Templates

Templates are a way to quickly add functionality to an item. If you want the player to be able to pick up an item, that will require a lot of attributes for the item. The TAKEABLE template does that for you.

Click the "Add" button to add a new template to this object. A new button will appear at the top, and you can use that to go to the section for attributes soecific to that template.

You are currently not able to remove a template.
